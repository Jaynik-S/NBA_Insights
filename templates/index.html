About NBA Insights

NBA Insights is a comprehensive web application designed for basketball enthusiasts, analysts, and fantasy players. It classifies NBA players into predefined archetypes using advanced data analytics and clustering algorithms, leveraging the nba_api for real-time player statistics.

Key features include player search, archetype identification, multi-archetype classification, interactive visualizations, and player comparison tools. With efficient data caching and intuitive visual analytics, NBA Insights provides a deeper understanding of player performance and trends across seasons.

This site is not affiliated with or endorsed by the National Basketball Association (NBA). All data is sourced from public APIs, including nba_api.
